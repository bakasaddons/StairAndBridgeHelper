# Stair And Bridge Helper

## How to Use the Addon

This addon is designed to assist users in creating stairs and bridges efficiently in their projects.

### Installation
1. Download the addon from the repository.
2. Import the addon into your project via your project's addon manager.
3. Activate the addon from the settings menu.

### Features
- Easy-to-use interface.
- Customizable options for stair types.
- Predefined bridge templates.

### Usage Instructions
- Access the addon from the toolbar after activation.
- Select the desired stair or bridge type from the menu.
- Customize dimensions and materials as needed.
- Click on the "Build" button to create the structure.

Happy Building!
